Knowledgebase

Frequently Asked Questions

How to start Prospero Application - You can start Prospero from the Start Menu. Clicking the Prospero Icon will open a new web page in your default internet browser. You will then be presented with the Prospero home page

Closing down Prospero - You can close Propsero like any other web page. This can be done at any time by clicking the x button on the browser window.

Prospero isn't working, what should i do...? - In the event that Prospero doesnt work, please perform the following steps:

  • close the application by closing the window.
  • Open a new browser window
  • Delete browser history and cookies by pressing the keyboard keys of Control + Shift + Delete. This will bring up a window that will allow you to clear your history.
  • Close down the browser window again.
  • Open Prospero again via the Start Menu

    • If Prospero is still not working please get in contact with Support at Support@fdmgroup.com You can also find more support details via the Support link in the Prospero Menu.

    Can I log into Prospero from more than one workstation ? - Yes, Prospero supports access from more than one workstation.

    I have a laptop/Deskop and mobile device. Which one can I use that will work with Prospero? Prospero supports multiple device types. These include Desktops, laptops, mobile phones and tablets. If you can access a web browser from the device than Prospero will work. Please note that Prospero will only work from an FDM office.

    Can I access Prospero from home, or away from the office? - No, Prospero will only work when you are in an FDM office. It can not be accessed from any other locations.
